Passgen

Version 1.3

About

Passgen is a Chrome extension to generate secure passwords using Secure Hash Algorithms.

  • You can set a master password every time you want and then enter an input to generate secure password.
  • Default hashing algorithm is set to SHA-256. You can change it whenever you want.
  • Tip: The default value of the input is set to the domain name of the current active tab but it is editable.
